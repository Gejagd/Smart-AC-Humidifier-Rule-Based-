# Smart AC-Humidifier

Program C++ untuk smart decision making AC & Humidifier.
tugas project smart decision making with rule based.

## Struktur
- include/      :  header file.
- src/          :  source code file.
- test/         :  trial-test code file.
- unit_test/    :  unit test file.

## Compile
- main program          :  g++ src/main.cpp src/smartdecision.cpp -Linclude -o app
- unit test program     :  g++ unit_test/unit_test.cpp src/smartdecision.cpp -Linclude -o testapp
